# Subscription Pricing Cards

This project is a simple subscription pricing layout built using HTML and CSS (Flexbox).

It includes three pricing plans with structured content, icons using Google Material Symbols CDN, and a responsive layout using Flexbox.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Google Material Symbols (CDN)

This project was created to practice layout alignment, spacing, and card structuring using core frontend concepts.
